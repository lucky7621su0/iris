# iris

This repository contains a script to generate a PowerPoint presentation for a BE project titled "Portfolio Optimization and Risk Analysis using Machine Learning and Streamlit".

Files:
- create_presentation.py: Generates a 10-slide PPTX with speaker notes and placeholder images.
- Portfolio_Optimization_and_Risk_Analysis_BE_Project.pptx: (Generated by running the script locally)

Usage:
1. Install dependencies:
   pip install python-pptx matplotlib numpy pandas seaborn
2. Run the script:
   python create_presentation.py

The script creates a PPTX and a folder `ppt_images` with sample charts. Replace placeholder images with real plots as needed.
